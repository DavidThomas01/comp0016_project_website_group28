Exploring vulnerabilities, mitigations and testing

Team 28 is working with Avanade on a security-focused project exploring prompt injection attacks, their impact on AI-driven applications, and how mitigations can be tested and applied—particularly in tool-connected systems such as Model Context Protocol (MCP) servers.

Project Overview

Prompt injection is an emerging attack vector where malicious instructions are embedded in user inputs to influence an AI model’s behaviour—potentially bypassing safeguards or causing the model to expose sensitive information.

Our project explores current prompt injection techniques, documents their impact, and evaluates mitigation strategies. A key focus is the security posture of MCP servers, which connect AI models to tools, APIs, and knowledge sources and may be exposed to injection vectors—especially when publicly accessible. :contentReference[oaicite:1]{index=1}

Objectives

  • Evaluate current prompt injection techniques and document their impact.
  • Evaluate mitigation strategies and their effectiveness.
  • Assess AI-application-specific vulnerabilities, with emphasis on MCP servers.
  • Design and implement a testing framework to assess MCP servers for susceptibility to prompt injection and related attacks (e.g., attempts to exfiltrate confidential data or API keys).
  • Investigate AI-driven automated testing for discovering and exploiting vulnerabilities.
  • Compare prompt injection security testing with traditional application security testing.
  • Produce technical recommendations and tooling approaches to help secure MCP servers against these threats.

Context

Prompt Injection Attacks

Prompt injection occurs when malicious instructions are embedded within user inputs to trick the model into ignoring intended instructions or revealing information it should not. Examples include attempts to extract system prompts or API keys, redirect actions to malicious endpoints, or bypass content and access controls. :contentReference[oaicite:3]{index=3}

Model Context Protocol (MCP) Servers

MCP is designed to standardise how AI models access external tools, APIs, and knowledge sources. MCP servers mediate access between the model and external systems—making them an important surface for security evaluation, particularly in public deployments. :contentReference[oaicite:4]{index=4}

Planned Deliverables

  • Proof-of-concept demonstrations of prompt injection techniques in realistic conditions.
  • A structured evaluation of mitigation strategies and practical guidance on when to use them.
  • A testing framework for assessing MCP servers against prompt injection and related exploits.
  • Technical recommendations and tooling approaches for securing MCP servers.
