GDPR & Data Privacy

This section documents data privacy considerations and GDPR compliance measures implemented in the platform.

🔒

Data Minimisation

The platform collects only the minimum data necessary for testing and does not store personal information.

💾

Local-First Storage

All test data is stored locally on the user's device with no transmission to external servers.

🗑️

Right to Erasure

Users can delete all stored test data and configurations at any time through the platform interface.

Privacy by design: The platform's local-first architecture inherently minimises data privacy risks by keeping all test data on the user's device. No persistent backend server means no centralised data collection.
